# mi_ayuno_intermitente

Un proyecto Flutter

En esta aplicación indicamos en que estado estamos del ayuno intermitente.
Fasting: Es el estado en que estamos ayunando, y que comienza con la última comida del día.
Feeding: Es el estado en que estamos alimentándonos, y que comienza con la primera comida del día.

La aplicación nos dice cuanto tiempo llevamos en cada estado.
